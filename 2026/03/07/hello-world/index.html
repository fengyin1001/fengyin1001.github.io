欢迎来到我的个人博客。

这是一个由 Hexo + GitHub Pages 搭建的技术博客，主要用于记录我在 网络安全学习道路上的笔记、思考和实践过程。

为什么创建这个博客？

学习网络安全是一条比较孤独的道路，很多时候需要自己查资料、自己实验、自己踩坑。

创建这个博客主要有三个目的：

  1. 记录学习过程
    把平时学习到的知识整理成文章，加深自己的理解。

  2. 分享技术
    希望我的文章能够帮助到同样在学习网络安全的朋友。

  3. 结识同行
    如果你也在学习网络安全，欢迎一起交流讨论，共同进步。

博客内容

这个博客主要会记录以下内容：

  • 网络安全知识
  • 实战复盘,漏洞分析与复现
  • 密码学
  • 数学
  • 个人感悟
  • 学习过程中的踩坑记录

更新频率

我会尽量保持 每周更新一篇文章，记录自己的学习进度。

关于我

一个正在努力学习网络安全的普通学生 / 技术爱好者。

目前主要学习方向：

  • 渗透测试
  • 数学
  • CTF

如果你有任何建议或者想一起交流，可以通过评论或者联系我。

Stay curious. Keep hacking.
